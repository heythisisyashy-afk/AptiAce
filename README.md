# 🧠 Aptitude Practice Test Platform

An interactive web-based aptitude practice platform built to help students prepare for placement exams and competitive tests.

🔗 **Live Demo:** https://sparkling-lolly-7ed76c.netlify.app/

---

## 📌 Overview

This project allows users to practice aptitude questions by simply pasting a JSON question set into the application. The platform automatically generates a timed aptitude test, evaluates the user's performance, and provides a detailed analysis to identify strengths and weaknesses.

It is designed for students preparing for placement exams such as **TCS, Infosys, Wipro, Accenture, Capgemini, Cognizant**, and other competitive examinations.

---

## ✨ Features

- 📄 Import aptitude questions using JSON format
- ⏱️ Timed online test
- 📝 Multiple-choice questions
- ⚡ Instant evaluation after submission
- 📊 Performance analysis
- 🎯 Topic-wise accuracy report
- 📈 Identify weak and strong areas
- 💡 Helps focus on topics that need improvement
- 📱 Responsive design for desktop and mobile

---

## 🚀 How It Works

1. Generate aptitude questions in **JSON format** using ChatGPT.
2. Copy the generated JSON.
3. Paste the JSON into the website.
4. Click **Start Test**.
5. Solve the questions within the time limit.
6. Submit the test.
7. View your score, detailed evaluation, and topic-wise analysis.

---

## 📋 JSON Format

The application accepts aptitude questions in JSON format.

Example:

```json
[
  {
    "id": 1,
    "question": "What is 15 + 25?",
    "options": ["30", "35", "40", "45"],
    "answer": "40",
    "category": "Quantitative Aptitude"
  }
]
```

You can easily generate this format using ChatGPT.
